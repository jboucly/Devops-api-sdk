## Description

SDK Typescript for interact with [Devops API](https://github.com/jboucly/Devops-API) from a Typescript front.

## Installation

```bash
$ npm i @jboucly/devops-api-sdk
```

Or if use yarn :

```bash
$ yarn add @jboucly/devops-api-sdk
```
